# 🌐 Azure Translator Web App

A simple and responsive web application that uses **Microsoft Azure Translator Service** to translate text into multiple languages. This project features a clean UI, dark/light mode toggle, and user-provided API configuration.

---

## 🚀 Features

* 🔑 Enter your own Azure API Key and Region
* 🌍 Translate text into multiple languages
* 🎨 Clean and responsive user interface
* 🌙 Dark / Light theme toggle
* ⚡ Fast and real-time translation using Azure API

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Microsoft Azure Translator API

## ⚙️ Setup Instructions

### 1. Clone the Repository

```
git clone https://github.com/your-username/azure-translator-app.git
cd azure-translator-app
```

### 2. Get Azure Translator API Key

* Go to Azure Portal
* Create a **Translator resource**
* Copy:

  * API Key
  * Region (e.g., `eastus`, `centralindia`)

---

### 3. Run the Project

Just open `index.html` in your browser:

```
double click index.html
```

OR use Live Server (recommended).

---

## 🌐 Usage

1. Enter your Azure API Key
2. Enter your Region (Location)
3. Type text in the input box
4. Select target language
5. Click **Translate**
6. View translated text instantly

---

## ⚠️ Important Note (Security)

This project uses the API key directly in the frontend.

👉 **Do NOT use this in production**

For production:

* Use a backend (Node.js / Python)
* Store API keys securely on the server
