Authorization checklist
  • Copy Key 1 or Key 2 from the resource's Keys and Endpoint page.
  • Use the exact resource location, for example eastasia.
  • Keep the endpoint as https://api.cognitive.microsofttranslator.com.
  • Check that the resource is active, quota is available, and public network access is allowed.
